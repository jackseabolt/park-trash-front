About

Trasher is a web application that facilitates the reporting of dump sites on public land. If a dump site is found, a user can create a public report of the issue with location information. That report is then publicly displayed, and either members of the public or local authorities can address each problem as it arises.

This application was constructed using Angular 4, Express, Node and Postgres. It was made in 2018 by Jack Seabolt.
